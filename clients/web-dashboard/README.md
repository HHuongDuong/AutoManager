# AutoManager Web Dashboard (React)

Ứng dụng dashboard theo định hướng React (Vite).

## Cài đặt

```bash
npm install
```

## Chạy phát triển

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Cấu hình nhanh

- API Base: mặc định http://localhost:3000
- Dùng tài khoản `/auth/login` để lấy token.
